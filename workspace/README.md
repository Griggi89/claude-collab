# Workspace

Shared code and output directory. All agents read from and write to this folder.

## Conventions
- Create a subdirectory per project: `workspace/project-name/`
- Don't modify another agent's active files without messaging them first
- Use clear commit messages: `feat:`, `fix:`, `wip:`, `docs:`
